# Face Analyser

Face Analyser is a real-time facial analysis system that detects a user's face through a webcam and predicts attributes such as emotions, age, facial expressions, and other visual characteristics. It provides instant insights by analyzing facial features and displaying the results through an interactive interface.

## Features
- Face Detection
- Emotion Recognition
- Age Estimation
- Facial Attribute Analysis
- Real-Time Webcam Processing
- Interactive Dashboard
